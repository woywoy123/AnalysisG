Overview

The Structs module contains fundamental data structures used throughout the framework.

Types

Key structures include:

  • settings_t: Analysis configuration
  • event_t: Event data container
  • graph_t: Graph data structure (see Graph Module)
  • folds_t: K-fold cross-validation data
  • optimizer_params_t: Optimizer configuration
  • model_report: Training progress reports

Property System

The module includes a property system (cproperty) for:

  • Type-safe getters and setters
  • Lazy evaluation
  • Python binding support
