Detailed Description

Defines the graph_template class and graph_t structure for graph-based analysis.

This file contains the declaration of the graph_template class and graph_t structure, which form the foundation of graph-based machine learning analysis in AnalysisG. The graph template defines how physics events are converted into graph representations, while graph_t holds the actual tensor data for training and inference.

Overview

Graph construction in AnalysisG follows this workflow:

  1. Define which particles become nodes via define_particle_nodes()
  2. Define edge topology via define_topology()
  3. Add node/edge/graph features using template methods
  4. Export to graph_t for batching and GPU transfer

Feature Types

Features are prefixed to indicate their purpose:

  • D-*: Data features (model inputs)
  • T-*: Truth features (training targets)
