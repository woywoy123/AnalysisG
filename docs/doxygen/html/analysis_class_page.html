Introduction

The analysis class is the central orchestrator for all physics analysis tasks in AnalysisG. It manages the complete workflow from data loading through model training to result evaluation.

Setup Phase

Add samples and templates:

// Add data samples
ana.add_samples("/data/ttbar/", "ttbar");
ana.add_samples("/data/background/", "bkg");
// Add event templates
ana.add_event_template(new MyEvent(), "ttbar");
ana.add_event_template(new MyEvent(), "bkg");
// Add graph templates
ana.add_graph_template(new MyGraph(), "ttbar");
ana.add_graph_template(new MyGraph(), "bkg");
// Add selection templates
ana.add_selection_template(new DileptonSelection());
// Add model for training
optimizer_params_t params;
params.lr = 0.001;
ana.add_model(new MyModel(), &params, "model_v1");
// Add metrics
ana.add_metric_template(new AccuracyMetric(), new MyModel());

  2. Project structure building
  3. Event processing

Internal Architecture

Internal Workflow

When start() is called, the following private methods execute:

  1. check_cache(): Look for cached data to skip reprocessing
  2. build_project(): Create output directory structure
  3. build_events(): Process raw data into event objects
  4. build_selections(): Apply selection criteria
  5. build_graphs(): Convert events to graph representations
  6. build_dataloader(): Prepare data for training
  7. build_model_session(): Configure model training sessions
  8. build_inference(): Set up inference pipeline (if applicable)
  9. build_metric(): Calculate performance metrics

Threading Model

The analysis uses a multi-threaded approach:

  • Event processing runs in parallel across files
  • Graph construction is parallelized per event
  • Model training uses batched parallel operations
  • Metric calculation can run concurrently with training

Examples

Basic Analysis

#include <AnalysisG/analysis.h>
int main() {
analysis ana;
// Configure
ana.m_settings.output_path = "results/";
ana.m_settings.threads = 4;
// Add data
ana.add_samples("/data/", "sample");
ana.add_event_template(new MyEvent(), "sample");
ana.add_graph_template(new MyGraph(), "sample");
// Run
ana.start();
return 0;
}

Training Example

#include <AnalysisG/analysis.h>
int main() {
analysis ana;
// ML configuration
ana.m_settings.epochs = 100;
ana.m_settings.kfolds = 5;
ana.m_settings.batch_size = 64;
// Setup
ana.add_samples("/data/signal/", "signal");
ana.add_samples("/data/background/", "bkg");
ana.add_event_template(new MyEvent(), "signal");
ana.add_event_template(new MyEvent(), "bkg");
ana.add_graph_template(new MyGraph(), "signal");
ana.add_graph_template(new MyGraph(), "bkg");
// Training configuration
optimizer_params_t params;
params.lr = 0.001;
params.weight_decay = 1e-5;
ana.add_model(new MyModel(), &params, "classifier");
// Run
ana.start();
return 0;
}
