Available Functions

The transform_ namespace provides coordinate transformations between Cartesian (px, py, pz, E) and polar (pT, η, φ, E) representations.

Polar → Cartesian

Function Parameters Returns Description
Px() pt, phi Tensor px = pT·cos(φ)
Py() pt, phi Tensor py = pT·sin(φ)
Pz() pt, eta Tensor pz = pT·sinh(η)
PxPyPz() pt, eta, phi Tensor[N,3] (px, py, pz)
PxPyPzE() pt, eta, phi, E Tensor[N,4] (px, py, pz, E)

Cartesian → Polar

Function Parameters Returns Description
Pt() px, py Tensor pT = √(px² + py²)
Phi() px, py Tensor φ = atan2(py, px)
Eta() px, py, pz OR pmc Tensor η = arctanh(pz/p)
PtEtaPhi() pmc OR px, py, pz Tensor[N,3] (pT, η, φ)
PtEtaPhiE() pmc OR px, py, pz, E Tensor[N,4] (pT, η, φ, E)

Example

#include <transform/transform.cuh>
// Convert from polar to Cartesian
torch::Tensor pt = torch::randn({100}).cuda();
torch::Tensor eta = torch::randn({100}).cuda();
torch::Tensor phi = torch::randn({100}).cuda();
torch::Tensor E = torch::randn({100}).cuda();
torch::Tensor pmc = transform_::PxPyPzE(&pt, &eta, &phi, &E);
// Convert back to polar
torch::Tensor pmu = transform_::PtEtaPhiE(&pmc);
