Overview

The Selection module provides the selection_template class for implementing event selection criteria.

Usage

To create a custom selection:

  1. Subclass selection_template
  2. Override the selection method
  3. Return true for events that pass the selection

Selection Chaining

Multiple selections can be combined:

  • Sequential (AND logic)
  • Named selections for cutflow analysis
