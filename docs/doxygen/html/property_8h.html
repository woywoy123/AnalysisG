Detailed Description

Defines the cproperty template class for property-based access patterns.

This file contains the declaration of the cproperty template class, which provides a property system similar to those found in languages like C# or Python. It allows defining getter and setter functions that are automatically called when the property is read or written, enabling validation, lazy evaluation, and computed properties.
