Detailed Description

Defines the io class for handling input/output operations, primarily with ROOT and HDF5 files.

This file declares the io class, which encapsulates functionalities for reading and writing data, particularly for physics analysis. It supports operations on ROOT TTrees and HDF5 datasets. The class inherits from tools and notification for utility functions and logging.
