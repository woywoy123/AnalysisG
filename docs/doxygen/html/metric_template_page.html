Metric Template Guide

Detailed guide for using and extending metric_template.

Introduction

The metric_template class defines performance metrics for evaluating model performance and analysis results.

Metric Types

Common metrics include:

  • Classification: Accuracy, precision, recall, F1
  • Regression: MSE, MAE, R²
  • Physics-specific: Mass resolution, efficiency

Example

class AccuracyMetric : public metric_template {
public:
AccuracyMetric() {
name = "accuracy";
}
void compute(graph_t* data, torch::Tensor prediction) override {
auto truth = *data->get_truth_node("label", this);
auto pred_class = (prediction > 0.5).to(torch::kInt);
auto correct = (pred_class == truth).sum().item<int>();
auto total = truth.size(0);
accuracy = (float)correct / total;
}
float accuracy;
};
