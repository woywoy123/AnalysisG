Container Reference

Documentation for the container class.

Introduction

The container class manages collections of events, graphs, and selections associated with a single input file or sample.

Structure

Each container holds:

  • Metadata: Sample information (cross-section, etc.)
  • Events: Processed event_template instances
  • Graphs: Constructed graph_template instances
  • Selections: Applied selection_template results

Entry Structure

The entry_t structure groups related objects:

struct entry_t {
std::string hash; // Event hash
std::vector<graph_t*> m_data; // Graph data
std::vector<graph_template*> m_graph; // Graph templates
std::vector<event_template*> m_event; // Events
std::vector<selection_template*> m_selection; // Selections
};
