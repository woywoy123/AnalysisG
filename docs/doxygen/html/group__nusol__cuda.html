Overview

The NuSol module implements algorithms for reconstructing neutrino momenta in semi-leptonic and di-leptonic top quark decays. This is essential for ttbar physics analyses.

Available Functions

Function Description
BaseMatrix() Compute base matrix for neutrino solutions
Intersection() Find intersection of constraint surfaces
Nu() Single neutrino reconstruction
NuNu() Double neutrino reconstruction (di-leptonic)
combinatorial() Combinatorial neutrino reconstruction

Algorithm

The neutrino reconstruction uses the following constraints:

  1. W boson mass constraint: m(lν) = m_W
  2. Top quark mass constraint: m(blν) = m_t
  3. Missing ET constraint: Σpx(ν) = MET_x, Σpy(ν) = MET_y

For single neutrino (semi-leptonic):

  • Uses W and top mass constraints
  • Solves quadratic equation for pz(ν)

For double neutrino (di-leptonic):

  • Uses both W mass constraints
  • Uses MET constraint
  • Solves system of equations iteratively

Example

#include <nusol/nusol.cuh>
// Semi-leptonic reconstruction
torch::Tensor pmc_b = ...; // b-jet 4-momentum
torch::Tensor pmc_mu = ...; // lepton 4-momentum
torch::Tensor met_xy = ...; // Missing ET (x, y)
torch::Tensor masses = ...; // [m_top, m_W, m_nu]
torch::Tensor sigma = ...; // MET resolution
auto result = nusol_::Nu(&pmc_b, &pmc_mu, &met_xy, &masses, &sigma, 1e-10);
torch::Tensor nu_momentum = result["nu"];
torch::Tensor chi2 = result["chi2"];

Combinatorial Reconstruction

For events with ambiguous jet-lepton assignments, the combinatorial() function tests all possible combinations:

auto result = nusol_::combinatorial(
&edge_index, &batch, &pmc, &pid, &met_xy,
172.62*1000, // m_top in MeV
80.385*1000, // m_W in MeV
1e-10, // null threshold
1e-3, // perturbation
100, // max steps
false // units (false = MeV, true = GeV)
