Detailed Description

Specialized modules extending AnalysisG functionality.

The Modules directory contains specialized components that extend the framework's capabilities. Each module provides specific functionality for physics analysis, machine learning, data handling, and utility operations.

Module Structure

Each module follows a consistent directory structure:

module_name/
├── CMakeLists.txt # Build configuration
├── include/ # Header files
│ └── templates/ # Template class headers
│ └── module.h
└── cxx/ # Implementation files
└── module.cxx
