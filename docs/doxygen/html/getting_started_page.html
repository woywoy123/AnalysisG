Getting Started with AnalysisG

A step-by-step guide to using the AnalysisG framework.

Overview

AnalysisG is a framework for graph-based analysis in high-energy physics. This guide walks you through the basic concepts and workflow.

Key Concepts

Events

An event represents a single physics collision. In AnalysisG, you define how to read event data by subclassing event_template.

Particles

Particles are physics objects within an event (electrons, muons, jets, etc.). Define particle types by subclassing particle_template.

Graphs

Graphs transform events into graph structures suitable for machine learning. Particles become nodes, and relationships become edges. Define graphs by subclassing graph_template.

Selections

Selections filter events based on physics criteria. Define selections by subclassing selection_template.

Models

Models are neural networks that process graph data. Define models by subclassing model_template.

Metrics

Metrics evaluate model performance. Define metrics by subclassing metric_template.

Typical Workflow

  1. Define your data structure
    • Create event template(s)
    • Create particle template(s)
  2. Define your graph representation
    • Create graph template(s)
    • Specify which particles become nodes
    • Define edge topology
    • Add features (data and truth)
  3. Define selections (optional)
    • Create selection template(s)
    • Implement selection criteria
  4. Define your model
    • Create model template
    • Implement forward pass
  5. Configure and run analysis
    • Create analysis instance
    • Add samples, templates, models
    • Configure settings
    • Start execution

Minimal Example

#include <AnalysisG/analysis.h>
#include <templates/event_template.h>
#include <templates/graph_template.h>
// Minimal event template
class SimpleEvent : public event_template {
public:
SimpleEvent() {
name = "simple_event";
}
};
// Minimal graph template
class SimpleGraph : public graph_template {
public:
SimpleGraph() {
name = "simple_graph";
}
void CompileEvent() override {
// Add features here
}
};
int main() {
analysis ana;
ana.m_settings.output = "output/";
ana.add_samples("/data/", "sample");
ana.add_event_template(new SimpleEvent(), "sample");
ana.add_graph_template(new SimpleGraph(), "sample");
ana.start();
return 0;
}
