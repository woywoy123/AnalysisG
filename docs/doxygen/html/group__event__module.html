Overview

The Event module provides the event_template class, which serves as the base class for defining event data structures in physics analyses.

Structure

An event template defines:

  • Trees: ROOT TTree names to read
  • Branches: Branch names within trees
  • Leaves: Individual variables to extract
  • Particles: Particle collections to construct

Creating Custom Events

To define a custom event type:

  1. Subclass event_template
  2. In the constructor, call add_tree(), add_branch(), add_leaf()
  3. Register particle types with register_particle()
  4. Override build(element_t*) to process raw data
