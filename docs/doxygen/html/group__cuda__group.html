Detailed Description

GPU-accelerated operations for physics computations.

The AnalysisG framework includes extensive CUDA support for accelerating computationally intensive physics operations. The CUDA modules are organized into namespaces that provide specific functionality.

Overview

The CUDA code is located in src/AnalysisG/pyc/ and provides GPU-accelerated implementations of:

  • Physics calculations (kinematics, invariant masses)
  • Coordinate transformations
  • Graph operations (aggregation, reconstruction)
  • Neutrino reconstruction (NuSol algorithm)
  • Mathematical operators (matrix operations, rotations)

Namespaces

PyTorch Integration

All CUDA operations are integrated with PyTorch tensors, allowing seamless GPU acceleration within the ML pipeline. Functions accept torch::Tensor* arguments and return torch::Tensor results.

Example usage in C++:

#include <physics/physics.cuh>
torch::Tensor pmc = ...; // [N, 4] tensor of (px, py, pz, E)
torch::Tensor mass = physics_::M(&pmc);
torch::Tensor beta = physics_::Beta(&pmc);
