Selection Template Guide

Detailed guide for using and extending selection_template.

Introduction

The selection_template class defines event selection criteria. Selections filter events based on physics requirements.

Example

class DileptonSelection : public selection_template {
public:
DileptonSelection() {
name = "dilepton";
}
bool selection(event_template* ev) override {
MyEvent* event = (MyEvent*)ev;
// Require at least 2 leptons
int nLeptons = event->electrons.size() + event->muons.size();
return nLeptons >= 2;
}
};

Chaining Selections

Multiple selections can be added to an analysis:

analysis ana;
ana.add_selection_template(new TriggerSelection());
ana.add_selection_template(new DileptonSelection());
ana.add_selection_template(new JetSelection());
