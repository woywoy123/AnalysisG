Detailed Description

Defines fundamental data structures and type handling for the AnalysisG framework.

This file contains declarations for basic structures and functions that handle data type identification, conversion, and management across the framework. It provides essential functionality for type safety and data management in a heterogeneous, physics data analysis environment.
