Available Functions

Function Parameters Returns Description

Beta Functions

Function Parameters Returns Description

Mass Functions

Function Parameters Returns Description
M2() px, py, pz, e OR pmc Tensor Invariant mass squared
M() px, py, pz, e OR pmc Tensor Invariant mass
Mt2() pz, e OR pmc Tensor Transverse mass squared
Mt() pz, e OR pmc Tensor Transverse mass

Angular Functions

Function Parameters Returns Description
Theta() pmc OR px, py, pz Tensor Polar angle θ
DeltaR() pmu1, pmu2 OR η1, η2, φ1, φ2 Tensor ΔR = √(Δη² + Δφ²)

Example

#include <physics/physics.cuh>
// Create momentum tensor [N, 4] = (px, py, pz, E)
torch::Tensor pmc = torch::randn({100, 4}).cuda();
// Calculate invariant mass
torch::Tensor mass = physics_::M(&pmc);
// Calculate beta factor
torch::Tensor beta = physics_::Beta(&pmc);
// Calculate DeltaR between particle pairs
torch::Tensor pmu1 = torch::randn({50, 4}).cuda();
torch::Tensor pmu2 = torch::randn({50, 4}).cuda();
torch::Tensor dR = physics_::DeltaR(&pmu1, &pmu2);
