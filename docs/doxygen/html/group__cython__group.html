Python interface to AnalysisG via Cython.

AnalysisG provides comprehensive Python bindings through Cython, allowing users to leverage the full power of the C++ framework from Python.

Overview

The Cython bindings are organized as follows:

Location Contents
src/AnalysisG/core/ Core class bindings (Analysis, templates)
src/AnalysisG/events/ Pre-built event templates
src/AnalysisG/graphs/ Pre-built graph templates
src/AnalysisG/models/ Pre-built neural network models
src/AnalysisG/selections/ Pre-built selection templates
src/AnalysisG/metrics/ Pre-built metric templates

File Structure

Each Cython module consists of:

  • .pxd file: C++ declarations (similar to header files)
  • .pyx file: Python implementation and bindings

Core Bindings

The core bindings in src/AnalysisG/core/ provide:
