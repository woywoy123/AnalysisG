Detailed Description

< Inherits from the notification class for logging capabilities.

Manages input and output operations for analysis data.

The io class provides an interface for reading from and writing to various data formats, with a focus on ROOT files (TTrees) and HDF5 files. It handles file opening/closing, data reading/writing for different data types (single objects, vectors of objects), and manages internal data structures for tracking file and dataset information.
