Particle representations and properties.

Overview

The Particle module provides the particle_template class for representing physics particles in the analysis.

Properties

Typical particle properties include:

  • Four-momentum (pT, eta, phi, E/m)
  • Identification (PDG ID, truth matching)
  • Quality flags
  • Isolation variables

Operations

Particles support operations like:

  • Four-vector addition
  • DeltaR calculations
  • Invariant mass computation
