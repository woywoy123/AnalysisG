Overview

The Graph module provides the graph_template class for constructing graph representations from physics events.

Graph Construction

To create a graph:

  1. Subclass graph_template
  2. In CompileEvent(), define which particles become nodes
  3. Define the edge topology using define_topology()
  4. Add features using add_node_data_feature(), add_edge_data_feature(), etc.

Feature Types

The graph template supports:

  • Node Data Features (D-*): Input features for nodes
  • Node Truth Features (T-*): Target labels for nodes
  • Edge Data Features (D-*): Input features for edges
  • Edge Truth Features (T-*): Target labels for edges
  • Graph Data Features (D-*): Input features for entire graph
  • Graph Truth Features (T-*): Target labels for graph
