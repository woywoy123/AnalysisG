Detailed Description

Provides a wide range of utility functions used throughout the framework.

The tools class serves as a toolkit, offering methods for file operations, string manipulations, container operations, type information retrieval, and more. Many classes in the AnalysisG framework inherit from tools to gain access to these common functionalities, promoting code reuse and consistency.
