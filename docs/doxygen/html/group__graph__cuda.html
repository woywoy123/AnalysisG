Usage

These functions are typically used for post-processing GNN outputs:

#include <graph/graph.cuh>
// After GNN forward pass
torch::Tensor edge_index = ...; // [2, E]
torch::Tensor prediction = ...; // [E] edge predictions
torch::Tensor features = ...; // [N, F] node features
auto result = graph_::edge_aggregation(edge_index, prediction, features);
torch::Tensor aggregated = result["aggregated"];
