IO Reference

Documentation for input/output operations.

Introduction

The io class handles all input/output operations in AnalysisG, supporting both ROOT and HDF5 file formats.

Supported Formats

Format Read Write Use Case
ROOT ✓ ✓ Physics data (ntuples)
HDF5 ✓ ✓ Graph data, tensors

ROOT Operations

Reading ROOT Files

io reader;
reader.trees = {"nominal"};
reader.branches = {"jet_", "el_", "mu_"};
reader.leaves = {"met_et", "met_phi"};
// Open and read
reader.start("data.root", "read");
reader.root_begin();
// Access data
auto* data = reader.get_data();
// Process entries...
reader.root_end();
reader.end();
