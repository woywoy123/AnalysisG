Detailed Description

Provides logging and messaging functionality with various severity levels.

The notification class serves as a base class for many components in the AnalysisG framework and provides a consistent interface for message output. It supports different severity levels (INFO, WARNING, ERROR, DEBUG) and can be configured to suppress certain types of messages or redirect output.
