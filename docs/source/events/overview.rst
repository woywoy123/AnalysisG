Events Package Overview
=======================

The events package contains concrete event implementations for different physics analyses.

Available Events
----------------

* **bsm_4tops**: BSM 4-top quark event implementation.
* **exp_mc20**: MC20 experimental event implementation.
* **gnn**: GNN inference event implementation.
* **ssml_mc20**: SSML MC20 event implementation.
