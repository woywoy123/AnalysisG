Installation
============

Instructions for installing AnalysisG.

Prerequisites
--------------

* C++ Compiler with C++17 support
* CMake (>= 3.14)
* ROOT Framework (optional)

Installation steps
--------------------

.. code-block:: bash

   git clone https://github.com/username/AnalysisG.git
   cd AnalysisG
   mkdir build && cd build
   cmake ..
   make
   make install
