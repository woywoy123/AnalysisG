=========
Changelog
=========

All notable changes to AnalysisG are documented here.

Version 1.0.0
=============

*Release Date: 2024*

Initial Features
----------------

- Core graph-based analysis framework
- Event template system for data reading
- Graph template system for graph construction
- Particle template system for physics objects
- Model template system for ML models
- Selection template system for event filtering
- Metric template system for performance evaluation
- PyTorch (LibTorch) integration
- ROOT file support
- HDF5 storage support
- Python bindings via Cython
- Multi-threaded processing
- K-fold cross-validation support
- Comprehensive documentation

Modules
-------

- **analysis**: Central orchestration
- **container**: Data container management
- **dataloader**: Efficient batching
- **event**: Event data structures
- **graph**: Graph construction
- **io**: Input/output operations
- **lossfx**: Loss functions
- **meta**: Metadata handling
- **metric**: Performance metrics
- **model**: ML model base
- **notification**: Logging system
- **optimizer**: Training loops
- **particle**: Particle representations
- **plotting**: Visualization
- **sampletracer**: Sample tracking
- **selection**: Event selection
- **structs**: Core structures
- **tools**: Utilities
- **typecasting**: Type conversions
