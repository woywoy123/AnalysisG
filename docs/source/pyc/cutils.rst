Cutils Component
================

Python-C++ interface for cutils operations.

File Location
~~~~~~~~~~~~~

* **Base Path**: ``src/AnalysisG/pyc/cutils/``
* **C++ Files**: ``src/AnalysisG/pyc/cutils/*.cxx``
* **CUDA Files**: GPU-accelerated implementations
* **Headers**: ``src/AnalysisG/pyc/cutils/include/**/*.h``

Description
-----------

This component provides high-performance cutils operations with optional GPU acceleration.

Features
--------

* CPU implementation in C++
* GPU implementation in CUDA
* Python interface through Cython
* Automatic device selection (CPU/GPU)
