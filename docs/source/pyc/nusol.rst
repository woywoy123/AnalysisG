Nusol Component
===============

Python-C++ interface for nusol operations.

File Location
~~~~~~~~~~~~~

* **Base Path**: ``src/AnalysisG/pyc/nusol/``
* **C++ Files**: ``src/AnalysisG/pyc/nusol/*.cxx``
* **CUDA Files**: GPU-accelerated implementations
* **Headers**: ``src/AnalysisG/pyc/nusol/include/**/*.h``

Description
-----------

This component provides high-performance nusol operations with optional GPU acceleration.

Features
--------

* CPU implementation in C++
* GPU implementation in CUDA
* Python interface through Cython
* Automatic device selection (CPU/GPU)
