Templates Package Overview
===========================

The templates package contains template files for creating new components.

Available Templates
-------------------

* **events**: Event template files
* **graphs**: Graph template files
* **metrics**: Metric template files
* **models**: Model template files
* **particles**: Particle template files

Description
-----------

Templates provide a starting point for implementing new:

* Event types
* Graph representations
* Evaluation metrics
* Machine learning models
* Particle types

Each template includes:

* Cython implementation file (.pyx)
* Cython header file (.pxd)
* C++ implementation file (.cxx)
* C++ header file (.h)
* CMakeLists.txt for building
* Markdown documentation
