Events Templates
================

Event template for creating new event types.

File Location
~~~~~~~~~~~~~

* **Template Path**: ``src/AnalysisG/templates/events/``

Files Included
--------------

* ``.pyx`` file: Cython implementation
* ``.pxd`` file: Cython header
* ``.cxx`` file: C++ implementation
* ``.h`` file: C++ header
* ``CMakeLists.txt``: Build configuration
* ``*.md`` file: Documentation template

Usage
-----

1. Copy template files to your module directory
2. Rename files appropriately
3. Implement required methods
4. Update CMakeLists.txt
5. Build and test
