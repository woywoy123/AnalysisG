Welcome to the AnalysisG documentation. AnalysisG is a sophisticated C++ framework designed for physics data analysis with integrated machine learning capabilities. The framework provides tools for event processing, graph-based data representation, and advanced machine learning models with PyTorch integration.

Features
--------

- **Event Processing**: Support for various event formats and physics objects
- **Graph-based Analysis**: Powerful graph structures for representing particle interactions and relationships
- **Machine Learning Integration**: Native support for training and inference with PyTorch models
- **Data Management**: Efficient data loading and batch processing capabilities
- **Physics Tools**: Specialized tools for physics calculations and transformations
- **Selection Workflow**: Modular system for event selection and categorization
- **Meta-data Handling**: Comprehensive meta-data tracking and utilization
